Los dominios en Internet

Luego de conocer la estructura de las direcciones WWW, cabe anotar, que para poder registrar el sitio web de una empresa y publicar páginas en Internet,

debe obtenerse un dominio. Este registro, contiene los últimos tres elementos mencionados en la estructura de una dirección web, y es el que permite que

diferentes usuarios, puedan encontrar un sitio web.
