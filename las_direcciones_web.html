Las direcciones Web

Una dirección Web, permite identificar en Internet, un servidor registrado, que está ubicado en algún sitio de la red. Esta dirección, no puede ser asignada a otro servidor; así el programa (protocolo) encargado de buscar una página, puede localizarlo rápidamente.

 

La estructura de las direcciones está formada por cinco partes, así:

Ejemplo: http://www.up.ac.pa

http:// Protocolo para transferencia de hipertexto (hyper text transfer protocol).

Es el software encargado de transmitir todo el código html, de las páginas web que viajan por Internet.

No es necesario digitar este segmento en la dirección del navegador, ya que este programa lo coloca automáticamente. www Indica que la dirección es una página en la World Wide Web.

 

Ejemplos:

http://www.prensa.com

Http://www.wikipedia.org/

 

Actualmente no todas las direcciones lo llevan.

 

Ejemplo: http://upmail.up.ac.pa

 

A partir de este componente, cada elemento se separa del otro por punto. up Contiene el nombre de la de la empresa o marca propietaria del sitio en

Internet

Ejemplos:

http://www.meduca.gob.pa/

http://www.icfes.gov.co/

http://www.iee.org.edu

http://www.fundacioncarolina.es

 

edu Identifica a la organización propietaria de la página. Hace referencia a la actividad a la cual se dedica la empresa

 

Ejemplos:

com ---------Comercial

mil -----------Militar

edu-----------Educativo

gov/gob---- Entidad Gubernamental

co y pa Indica el país donde se encuentra ubicado o registrado el servidor.

Ejemplos:

http://www.uptc.edu.co    Colombia

http://www.msn.es     España

http://www.physicaleducation.co.uk/   Reino Unido

http://www.utp.ac.pa  Panamá

 

Generalmente los sitios registrados en Estados Unidos no utilizan este componente, como:

http://www.whitehouse.gov

http://www.yahoo.com

 

Esta estructura no es obligatoria y muchos sitios han eliminado algunos componentes; sin embargo, en un gran porcentaje de esta estructura se sigue conservando.
